<h1>🌾 무트코인 (MootCoin) — 농산물 가격 예측 & 병충해 탐지 시스템</h1>

무트코인은 농가의 소득 안정과 지속 가능한 농업을 목표로 하는 통합 예측 시스템입니다.
기상·시장 데이터를 기반으로 한 무 가격 예측 모델과, 실제 무 작물의 이미지 기반 병충해 탐지 모델을 결합해
농업인, 유통업자, 소비자 모두에게 실질적인 도움을 제공하고자 설계되었습니다.

<h1>📌 프로젝트 개요</h1>

프로젝트명: 무트코인 (MootCoin)

참여 팀: 3조 쌈무

기간: 2023.1 – 2023.12

주요 목표:

무 가격 예측을 통해 판매/소비 시기 예측

병해충 탐지를 통한 생산 안정성 확보

농산물 유통 시장의 합리적인 의사결정 기반 제공

<h1>🧠 핵심 기능 요약</h1>

<h3>기능	설명</h3>

📈 가격 예측	GRU 기반의 시계열 모델로 무의 가격을 예측 (소/도매 유통사와 소비자용 모델 구분)

🐛 병충해 탐지	YOLOv7 기반 이미지 인식 모델로 무 잎의 병해충 여부를 실시간 분석

🧹 전처리 자동화	기상청(KMA), KAMIS 데이터를 수집 후 병합, 누락 보간 및 정규화 처리

🌐 프론트엔드	Visual Studio Code 기반 웹 페이지에서 예측 결과 시각화 기능 구현

<h1>🧪 사용 데이터 및 기술</h1>

<h3>📊 데이터 출처</h3>

무 가격 데이터: 한국농수산식품유통공사 (KAMIS)

기상 데이터: 기상청 기상자료개방포털 (일 평균 기온, 강수량)

병해충 이미지: AI-Hub (노지 작물 질병 진단 이미지, 약 12,000장)

<h1>🛠 기술 스택</h1>

<h3>분야	기술</h3>

가격 예측	GRU, Bi-GRU, Transformer, Scikit-learn, PyTorch

병충해 탐지	YOLOv7, Label 변환(JSON → YOLO format), OpenCV

프론트엔드	HTML/CSS + JavaScript, VS Code

데이터 처리	Pandas, NumPy, MinMaxScaler, 시계열 정제

<h1>📈 모델 성능 요약</h1>

<h3>📉 가격 예측 (GRU 기준)</h3>

MSE: 0.2

R² Score: 0.45

양방향 GRU를 적용하여 기온/강수량의 순방향 + 역방향 영향까지 학습

<h1>🌿 병충해 탐지 (YOLOv7 기준)</h1>

AI-Hub에서 제공된 라벨링 데이터를 변환하여 학습

다양한 병충해 클래스(class)와 무 잎 사이의 시각적 차이를 높은 정확도로 분류
